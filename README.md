# Swiftensions

`Swiftensions` is a little handy library of Swift extensions.


## Requirements
- Swift 3.0.2 / Xcode 8.2.1
- iOS 8.0 and later


## Extensions

### Foundation
- Array
- Bundle
- Calendar
- IndexPath
- NSNumber
- Optional
- Sequence
- String
- URL

#### UIKit
- UIDevice
- UINib
- UIView
