# Swiftensions

Swiftensions は Swift の Extension をまとめたライブラリです。
iOS標準ものなど、ちょっと便利な機能を提供します。

## Requirements
- Swift 3.0.2 / Xcode 8.2.1
- iOS 8.0 and later

## 内容
### Foundation
- Array
- Bundle
- Calendar
- IndexPath
- NSNumber
- Optional
- Sequence
- String
- URL

#### UIKit
- UIDevice
- UINib
- UIView
